Christian Koller

Christian ist ebenso wie Christoph Wille gebürtiger Oberösterreicher, allerdings wird man seiner die meiste Zeit woanders habhaft, sei dies in Wien, Leoben, oder auf seinem Motorrad. Seine Karriere in der Webserverprogrammierung hat Christian auf Veranlassung von Christoph gestartet, der meinte, einen guten Datenbankprogrammierer zu brauchen - und somit ist er Ende 97 in das Internetgeschäft eingestiegen.

Christian's Spezialgebiete sind SQL Server Programmierung sowie ASP Optimierung, und dieses Wissen hat er sichunter anderem im Laufe eines Immobilienserverprojektes in den USA angeeignet. Auch Komponentenprogrammierung schreckt ihn nicht ab, allerdings die ATL/C++ Projekte bleiben dann doch für Christoph reserviert.

Er war es, der im Teach Yourself ASP in 14 Hours Buch unter anderem die Kapitel über ADO übernommen hat. Im letzten gemeinsamen Buchprojekt mit Christoph, Teach Yourself ADO 2.5 in 21 Days, hat er mit ziemlich genau der Hälfte der Kapitel einen wichtigen Part geleistet.

Christian kann beim Schreiben seiner Artikel auf seine Erfahrung mit folgenden Technologien zurückgeifen:


Active Server Pages (ASP): Planung von ASP-Websites, Implementation und Entwicklung von ASP-Applikationen unter VBScript und JScript, E-Commerce mit Datenbankanbindung unter ADO 2.5 auf SQL-Server 7.0 und Access 2000, Einsatz kommerzieller ASP-Komponenten (AspQMail, ASPTear, BrowserHawk 2000, u.a.), Performance Tuning, Webserver-Security, und Server/Client-Applikationen, Massemmails über CDO/NTS.

ADO 2.5 (ActiveX Data Objects): Universeller Datenzugriff unter Microsoft Betriebssystemen, ADODB & ADOX Libraries, Datenbankzugriff über ODBC und OLE DB auf Datenquellen wie Datenbankserver oder Index-Server über Intra- & Internet.

SQL-Server 7.0: Entwicklung und Implementation von Datenbanken unter SQL Server 7.0, Programmierung von Stored Procedures mit t-SQL, Administration von SQL-Servern und -Datenbanken mittels SQL Server Enterprise Manager (auch über Internet).

Webserver: Erfahrung in der Konzeption, Umsetzung und dem Betreiben von Websites unter Internet Information Server (IIS) 4.0 & 5.0 (Windows NT 4.0 Server und Windows 2000 Server), sowie unter Personal Web Server (PWS) auf Windows 95/98.

HTML 4.0: Programmierung von Internetseiten (statisch und dynamisch), Browserkompatibilität (Internet Explorer, Netscape, Opera, u.a.).

Client-side JavaScript: Für browserseitige Ergänzungen von ASP-Seiten und Standalone-Applikationen, Browserkompatibilität von JavaScript-Seiten.

Englisch in Sprache und Schrift: Zusammen mit Christoph Wille Autor folgender englischsprachigen Bücher des SAMS Verlages (USA):

  • Teach Yourself ASP in 24 Hours
  • Teach Yourself ADO 2.5 in 21 Days
