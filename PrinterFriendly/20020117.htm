Last but not least - .NET 1.0 ist da!

Geschrieben von: Christoph Wille
Kategorie: .NET Allgemein

This printed page brought to you by AlphaSierraPapa

Lange hat es gedauert - aber seit Mittwoch früh (US Zeit) ist .NET 1.0 zum Download verfügbar. Neben den Links zu den Downloads (SDK und ASP.NET Releases) finden sich in diesem Artikel auch Links zu Sites rund um die verschiedenen in .NET enthaltenen Technologien, als auch zu Diskussionsforen, in denen man sich mit anderen Programmierern austauschen kann.

  • Download von (ASP).NET 1.0
  • Diskussionslisten
  • Links zu ASP.NET
  • Links zu C#
  • Weitere wichtige Links

Downloads

Die wichtigsten Links zum Download von ASP.NET und .NET:

  • Das vollständige .NET 1.0 SDK (empfohlener Download für Entwickler)
  • ASP.NET 1.0 (empfohlen zur Installation von ASP.NET am Server)
  • Die .NET 1.0 Distributables (alles was ein "normaler" User für .NET Anwendungen braucht)

Diskussionslisten

Eine - klarerweise nicht vollständige - Liste von Diskussionsforen zu .NET, ASP.NET und den Programmiersprachen. Die deutschen Foren sind am Anfang aufgelistet.

  • C# Diskussionsliste
  • VB.NET Diskussionsliste

  • ASP.NET Listen (Englisch; viele ASP.NET Team Member nehmen an den Diskussionen teil)
