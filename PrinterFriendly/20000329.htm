ADO Component Checker Tool

Geschrieben von: Christian Koller
Kategorie: Datenbank

This printed page brought to you by AlphaSierraPapa

Mit Hilfe des Component Checker Tools kann man herausfinden, welche ADO (ActiveX Data Objects) und MDAC (Microsoft Data Access Components) Version auf einem Computer installiert ist. Das Component Checker Tool ist Teil des Platform SDK, kann aber auch von der Microsoft Website separat downgeloaded werden.

Um das Component Checker Tool laufen zu lassen, benötigen sie ein 32-bit oder 64-bit Betriebssystem, wie Windows 98, Windows Millennium, Windows NT 4.0 und Windows 2000.

Die folgenden Aufgaben kann das Component Checker Tool übernehmen:

  • Identifikation der im Augenblick installierten MDAC und ADO Version eines Betriebssystems.
  • Auflisten der Dateien der installierten MDAC Version, erstellen einer Aufstellung der Unterschiede zu größeren MDAC Release Versionen.
  • Komplettes Deinstallieren von MDAC (und damit auch ADO). Zusätzlich werden DLL Konflikte und Programme die bestimmte DLLs benutzen aufgelistet.

Zum Installieren des Component Checker Tools einfach das Setup Programm (cc.exe) laufen lassen und das Tool in ein Verzeichnis installieren (C:\comcheck) . Um das Component Checker Tool zu aktivieren, die Datei ComCheck.exe ausführen.

Das Tool erlaubt 3 verschiedene Arten der Analyse:

  • Genaue Analyse der MDAC Installation und Bestimmung der installiereten MDAC Release Version.
  • Analyse der MDAC Installation und Vergleich mit einer ausgewählten MDAC Version.
  • Nur Scannen der installierten MDAC, OLE DB, ODBC und ADO Dateien.

Da das Component Checker Tool alle MDAC Dateien mit Versionsnummern ausgibt, kann man auch ermitteln welche Version eines bestimmten OLE DB Providers auf dem System installiert ist.

Um beispielsweise die Versionsnummer des installierten OLE DB Providers for Jet herauszufinden, starten sie das Component Checker Tool und führen einen Scan der MDAC Dateien durch. Danach öffnen sie den OLEDB Ordner, indem sie auf das Plus Zeichen vor dem OLEDB Icon im linken Teil des Fensters klicken. Im geöffneten Teil klicken sie auf Microsoft Jet x.x OLE DB Provider und sie sehen im rechten Fensterteil den vollständigen Versionsnamen des Providers:

Um die Dokumentation des Tools zu aktivieren, klicken sie im Menü auf Help, Help topics. In der Hilfe finden sie auch eine ausführliche Anleitung wie man eine MDAC Installation komplett de-installieren kann, was zum Beispiel nötig ist, um eine ältere MDAC Version installieren zu können.
