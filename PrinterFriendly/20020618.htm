Das ASP.NET Web Matrix Projekt

Geschrieben von: Christoph Wille
Kategorie: ASP.NET

Nach einer längeren Testphase (und nach einigen Gerüchten) ist es nun so weit - die ASP.NET Web Matrix ist endlich da! Hinter dem Namen ASP.NET Web Matrix verbirgt sich ein freies, leicht zu erlernendes Entwicklungstool für ASP.NET Applikationen, das selbst mit Hilfe von .NET, C# und Windows Forms implementiert wurde. Web Forms und Web Services bequem entwickeln ohne Visual Studio .NET kaufen zu müssen ist damit Realität.

Das wichtigste vorweg - der Downloadlink für die ASP.NET Web Matrix.

Das zweitwichtigste folgt auf dem Fuß - der Link zur Guided Tour. Obwohl in Englisch gehalten, ist das gleichzeitig der beste Start als auch Überblick über die ASP.NET Web Matrix. Man ist damit sofort produktiv, und erspart mir eine detaillierte Ausführung der Features, die da wären (unter anderem):

  • Ein graphischer Designer für ASP.NET Web Forms
  • Unterstützung für Web Services
  • Integrierter Web Server zum Testen der gerade erstellten Web Forms und Web Services. Es muß lokal kein IIS installiert sein, was dieses Feature für viele Entwickler herausheben wird.
  • Mehrsprachigkeit (C# und Visual Basic.NET)
  • Projektmanagement (Workspaces) ohne Frontpage Server Extensions
  • Datenbankmanagement für MSDE und SQL Server

Gerade letzteres Feature wird für viele Entwickler eine Erleichterung darstellen, da die MSDE ja ohne graphischem Administrationsfrontend ausgeliefert wird. Der passende Teil der Guided Tour ist Creating a Database.
